# Fire Ninja — second example

A simpler character than the Wizard, showing a small `effects` list (three
outcomes) and a shirt tint (`bodyColor`) on the Assassin's body.

- **Fast & accurate**: Speed 4, Accuracy 4.
- **Shadow Flame**: throws a fireball, or turns invisible, or blinks away.
- **Big Quest — "Way of the Flame"**: defeat 6 enemies with Shadow Flame.
