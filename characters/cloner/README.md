# Cloner

Cloner is super fast but super fragile — a glass cannon on the run. Their special
power, **Copycat**, lets them aim at any piece of furniture — a chair, a wall, a
vending machine — and copy it, dropping a brand-new duplicate right beside it.
Build yourself some cover, block a doorway, or just make a giant mess! It recharges
every 15 seconds. Cloner wears the spooky Vampire look with a bright green outfit.
Their Big Quest, **Hall of Mirrors**, is to copy 12 objects in a run — no fighting
required. (It's built with a custom code quest in `src/CloneQuest.cs`, a mission
type the base game doesn't have.)
